\section{Introducción}

Este documento contiene los diagramas técnicos regenerados para el Manual Técnico de Lubricación Industrial de AISA 2026. Todos los diagramas han sido creados en formato vectorial TikZ, garantizando:

\begin{itemize}
    \item \textbf{Escalabilidad}: Los diagramas pueden ampliarse sin pérdida de calidad.
    \item \textbf{Consistencia}: Paleta de colores corporativa AISA aplicada uniformemente.
    \item \textbf{Editabilidad}: Código fuente disponible para futuras modificaciones.
    \item \textbf{Precisión}: Puntos de lubricación claramente identificados con frecuencias.
\end{itemize}
